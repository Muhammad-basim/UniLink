# UniLink – Smart University Assistant

A console-based university management system developed in C for the
Programming Fundamentals semester project.

## Features
- Admin login
- Student management (add/update/delete/search)
- Course management
- Student–course assignment
- Teacher–course assignment (in progress)

## Technologies
- C language
- File handling (CSV)
- Makefile
- Dev-C++
